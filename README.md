## Overview

This is a modern and responsive personal portfolio website developed to showcase my projects, technical skills, certifications, achievements, and professional experience. The portfolio is designed with a clean user interface and smooth user experience to create a strong professional online presence.

The project demonstrates frontend development skills, responsive design implementation, routing structure, reusable component architecture, and modern web development practices using React and Vite.

---

# Features

* Modern and responsive UI/UX
* Reusable React components
* Multi-page routing structure
* Project showcase section
* Skills and technology display
* Contact and social links integration
* Optimized performance using Vite
* Mobile-friendly design
* Clean folder architecture

# Installation & Setup

## Clone the Repository

```bash id="mqrvto"
git clone https://github.com/Shubhendu-Singh26/portfolio.git
```

## Navigate to the Project Directory

```bash id="59y2n4"
cd your-portfolio
```

## Install Dependencies

```bash id="3eqh2v"
npm install
```

## Run the Development Server

```bash id="6snv13"
npm run dev
```

The application will start locally at:

```bash id="l6lv6j"

```
